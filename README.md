  其次，什么是线性规划问题，以及选择什么工具来解决这类问题，希望我的另一个文章线性规划和约束满足问题http://blog.csdn.net/yin_wuzhe/article/details/52563109（）能很好的解给出我的解答。
  最后，我将总结一些使用的基本步骤，和对代码做出必要性的解释。
